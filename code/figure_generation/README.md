# figure_generation/

Scripts for generating publication figures.

## Contents

- `generate_6k_vs_12k_performance.py`: Generate comparison of 6K vs 12K claim performance
- `generate_base_model_performance_comparison_one_column.py`: Generate single-column base model performance figure
- `generate_base_model_performance_comparison.py`: Generate base model performance comparison figure
- `generate_citation_type_and_top_domains_one_column.py`: Generate single-column citation analysis figure
- `generate_citation_type_and_top_domains.py`: Generate citation type and top domains figure
- `generate_combined_performance_heatmaps.py`: Generate combined performance heatmap figures
- `generate_leaning_lin10k_jointplot.py`: Generate political leaning vs Lin quality jointplot
- `generate_leaning_newsguard_jointplot.py`: Generate political leaning vs NewsGuard quality jointplot
- `generate_macro_weighted_correlation.py`: Generate macro/weighted metric correlation figure
- `generate_nei_plot.py`: Generate "Not Enough Information" analysis figures
- `generate_reasoning_web_vs_base_performance_one_column.py`: Generate single-column reasoning/web comparison figure
- `generate_reasoning_web_vs_base_performance.py`: Generate reasoning and web search vs base performance figure
