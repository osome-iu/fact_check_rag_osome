# data_collection/

Scripts for collecting and processing PolitiFact data.

## Contents

- `001_scrape_politifact_analysis_text.py`: Scrape fact-check analysis article text from PolitiFact
- `002_summarize_analysis_text.py`: Use LLM to summarize PolitiFact article text for RAG
- `politifact_scraper/`: Local package and scripts for scraping PolitiFact claims
