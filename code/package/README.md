# package/

Local Python package containing shared utilities for the fact-checking project.

## Contents

- `llm_fact/`: Core package module containing prompts, response schemas, metrics, and utilities
- `setup.py`: Package installation configuration

## Installation

After moving to this directory, install the package locally with pip. The `-e` flag enables editable mode, meaning any changes are immediately available when re-imported.

```sh
pip install -e ./
```
