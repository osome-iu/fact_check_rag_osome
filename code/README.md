# code/

All code for the project is stored here.

## Contents

- `build_dbs/`: Scripts for building the ChromaDB vector database for RAG
- `data_analysis/`: Scripts for analyzing results and generating performance metrics
- `data_cleaning/`: Scripts for cleaning and processing data for analysis
- `data_collection/`: Scripts for collecting and processing PolitiFact data
- `factchecking_tests/`: Scripts for running fact-checking experiments across LLM providers
- `figure_generation/`: Scripts for generating publication figures
- `notebooks/`: Jupyter notebooks for exploratory analysis and development
- `package/`: Local Python package with shared utilities
- `00-db-collection-and-generation-pipeline.sh`: Pipeline for scraping data and building RAG database
- `01-run-factchecking-tests.sh`: Pipeline for running LLM fact-checking tests
- `02-data-analysis-pipeline.sh`: Pipeline for cleaning and analyzing data
- `03-generate-results-and-figures.sh`: Pipeline for generating tables and figures
