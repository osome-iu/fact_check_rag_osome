# factchecking_tests/

Scripts for running fact-checking experiments across multiple LLM providers.

## Contents

- `002_test_gemini_models.py`: Run fact-checking tests using Google Gemini models
- `002_test_openai_models.py`: Run fact-checking tests using OpenAI models
- `002_test_togetherai_models.py`: Run fact-checking tests using Together AI models
- `003_generate_openai_reasoning_batches.py`: Generate batch files for OpenAI reasoning model tests
- `003_submit_and_monitor_openai_batches.py`: Submit and monitor OpenAI batch API jobs
- `test_gemini_all.sh`: Shell script to run all Gemini model configurations
- `test_openai_all.sh`: Shell script to run all OpenAI model configurations
- `test_together.sh`: Shell script to run all Together AI model configurations
- `XXX_delete_all_openai_batch_files.py`: Utility to delete OpenAI batch files
